message - Configurations and customizations for the message

Configurations and customization to send an email, through Gmail, so a mobile phone will receive it as a MSM
Properties:
Name Type Description
displayName string Will not appear in the MMS message but will be the sender name in the email copy located in the sender's gmail
sendAs string Email address that the receiver will see as the sender. Must be the Gmail accounts main email address OR and alias
sendTo string Receiver's mobile phone number followed by their carrier's MMS gateway address
subject string Subject displayed at the top of the message
bodyTemplateName string File name of the HTML template used to structure the message body
